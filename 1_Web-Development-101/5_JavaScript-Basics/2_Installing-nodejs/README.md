# [Installing Node.js](https://www.theodinproject.com/lessons/foundations-installing-node-js)
**See above link for instructions:* `nvm` and `node` already installed on `lam`* 

## Set `lts` version
```bash
# install and set
nvm install --lts
nvm use --lts # set most recent LTS version as default
```

### Verify
```bash
npm -v
nvm list
node -v
```
